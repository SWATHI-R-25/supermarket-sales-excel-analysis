# Supermarket Sales Analysis (Excel)

## Project Overview
This project analyzes supermarket sales data using Microsoft Excel.
The objective is to understand sales trends, customer payment preferences,
product performance, and branch-wise sales.

## Dataset
- Supermarket Sales Dataset (Kaggle)
- Data covers 3 months of sales transactions

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Dashboard

## Analysis Performed
- Monthly Sales Analysis
- Sales by Product Line
- Sales by Payment Method
- Sales by Branch

## Key Insights
- January recorded the highest sales.
- Cash is the most commonly used payment method.
- Branch C generated the highest total sales.
- Product lines contributed differently to total revenue.

## Dashboard
The Excel dashboard visually summarizes all insights using interactive charts.

## Conclusion
This project demonstrates practical Excel skills including data analysis,
visualization, and dashboard creation for business decision-making
